GitOps for Infrastructure

인프라 변경을 Git 워크플로우로 관리합니다. PR 기반으로 변경사항을 리뷰하고 자동화된 파이프라인이 Plan/Apply를 수행합니다.

Atlantis

PR 기반 Terraform 자동화 도구입니다. Webhook으로 PR 이벤트를 감지하고 자동으로 Plan을 실행하여 코멘트로 결과를 보여줍니다.

Spacelift

Terraform/OpenTofu 전용 CI/CD 플랫폼입니다. Policy as Code, Drift Detection, Stack Dependencies 등 고급 기능을 제공합니다.

Plan Diff 읽기

+ (녹색) = 리소스 생성, ~ (노란색) = 리소스 수정, - (빨간색) = 리소스 삭제. Apply 전 반드시 Plan 결과를 확인해야 합니다.
