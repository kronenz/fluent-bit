Remote State Backend

S3+DynamoDB 조합이 AWS 환경의 표준입니다. 상태 파일을 S3에 저장하고 DynamoDB로 동시 접근을 방지합니다. 암호화, 버전관리가 필수입니다.

State Locking 상태 잠금

terraform plan/apply 실행 시 자동으로 Lock을 획득합니다. 다른 사용자의 동시 변경을 방지하여 State 충돌을 예방합니다.

Drift Detection 드리프트 감지

콘솔에서 수동 변경 시 State와 실제 인프라가 불일치합니다. terraform plan이 이를 감지하고 코드 상태로 되돌리거나 코드를 업데이트합니다.

terraform import

기존 리소스를 Terraform 관리로 가져옵니다. 먼저 .tf 코드를 작성하고 import 명령으로 State에 매핑합니다. Terraform 1.5+에서는 import 블록을 지원합니다.
