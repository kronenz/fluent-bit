VPC Architecture

Multi-AZ 구성으로 고가용성을 확보합니다. Public Subnet에 ALB/NAT GW, Private Subnet에 Worker Node를 배치합니다. CIDR 설계가 중요합니다.

EKS Control Plane

AWS가 완전 관리하는 Kubernetes Control Plane입니다. API Server, etcd, Controller가 Multi-AZ로 HA 구성됩니다. 업그레이드도 AWS가 처리합니다.

IRSA (IAM Roles for Service Accounts)

Pod 단위로 최소 권한 IAM Role을 부여합니다. OIDC Federation을 통해 ServiceAccount와 IAM Role을 매핑합니다. Node Role 공유보다 안전합니다.

Managed Node Group

AWS가 EC2 인스턴스 수명주기를 관리합니다. Auto Scaling, 자동 AMI 업데이트, Graceful Drain을 지원합니다. Spot Instance도 혼합 가능합니다.
