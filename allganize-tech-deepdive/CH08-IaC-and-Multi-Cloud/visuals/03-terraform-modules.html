Root Module 루트 모듈

terraform apply를 실행하는 디렉토리가 Root Module입니다. Child Module을 호출하고 변수를 전달하며 Output을 수집합니다.

Child Module 자식 모듈

재사용 가능한 인프라 단위입니다. source로 Registry, GitHub, 로컬 경로를 지정할 수 있습니다. version으로 특정 버전을 고정합니다.

Input Variables 입력 변수

variable 블록으로 정의하며, 모듈 호출 시 값을 전달합니다. type, default, validation, description으로 안전하게 설계합니다.

Output Values 출력 값

output 블록으로 모듈의 결과를 외부에 노출합니다. 다른 모듈의 입력으로 연결하거나 CLI에서 확인할 수 있습니다.
