Step 1. sysctl 파라미터 (핵심 튜닝 항목)

net.core.somaxconn
Listen backlog 최대 크기 (TCP 연결 대기열)
기본: 128튜닝: 65535
net.ipv4.tcp_tw_reuse
TIME_WAIT 소켓 재사용 허용
기본: 0 (OFF)튜닝: 1 (ON)
vm.swappiness
Swap 사용 적극성 (0=최소, 100=적극)
기본: 60튜닝: 10
vm.overcommit_memory
메모리 오버커밋 정책 (0=휴리스틱, 1=항상, 2=제한)
기본: 0튜닝: 0 또는 2
net.ipv4.ip_local_port_range
로컬 포트 범위 (클라이언트 연결 시)
기본: 32768-60999튜닝: 1024-65535
fs.file-max
시스템 전체 파일 디스크립터 최대 수
기본: ~100000튜닝: 2097152
