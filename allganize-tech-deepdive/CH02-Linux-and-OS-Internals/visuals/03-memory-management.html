Step 2. Page Fault 처리 (Minor vs Major)

⚡ Minor Page Fault

1. 페이지 접근 시도 2. 페이지 테이블에 매핑 없음 3. 페이지가 이미 메모리에 존재 (캐시) 4. 페이지 테이블 업데이트 → 완료 ✔ 디스크 I/O 없이 빠르게 해결

💾 Major Page Fault

1. 페이지 접근 시도 2. 페이지 테이블에 매핑 없음 3. 메모리에도 없음 → 디스크 읽기 필요 4. 디스크에서 페이지 로드 (느림!) ⚠ 디스크 I/O 발생 → 성능 영향 큼
