Access Modes: 볼륨 접근 방식 비교

RWO

ReadWriteOnce
단일 노드에서 읽기/쓰기

R/W

EBS, Azure Disk
단일 Pod DB에 적합

RWX

ReadWriteMany
다수 노드에서 읽기/쓰기

R/WR/WR/W

NFS, EFS, CephFS
공유 파일 업로드에 적합

ROX

ReadOnlyMany
다수 노드에서 읽기 전용

RRR

설정 파일 공유
정적 자산 서빙에 적합

Reclaim Policy: PVC 삭제 후 PV 처리 방식

Retain (보존)

1. PVC 삭제 요청
2. PV 상태: Released
3. 데이터 보존됨 ✓
4. Admin이 수동 정리
5. PV 재사용 또는 삭제

프로덕션 DB에 권장
데이터 유실 방지

Delete (삭제)

1. PVC 삭제 요청
2. PV 자동 삭제 시작
3. Cloud Volume 삭제
4. 데이터 영구 삭제 ✗
5. 스토리지 비용 절감
