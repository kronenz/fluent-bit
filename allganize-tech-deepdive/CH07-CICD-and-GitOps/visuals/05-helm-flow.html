OVERRIDE Values 우선순위 Override Hierarchy

높은 우선순위가 낮은 우선순위를 덮어씀 (Higher priority overrides lower)

1

--set / --set-string (최고 우선순위 Highest)

CLI에서 직접 값 지정. CI/CD 파이프라인에서 동적 값 전달에 유용

helm install my-app ./chart --set image.tag=v2.0.0
⬇ overrides
2

-f / --values (사용자 Values 파일)

환경별 values 파일로 오버라이드. 여러 파일 지정 가능 (뒤가 우선)

helm install my-app ./chart -f values-prod.yaml
⬇ overrides
3

values.yaml (차트 기본값 Chart Defaults)

차트에 포함된 기본 값. 모든 사용자에게 공통 적용되는 베이스라인

# chart/values.yaml (defaults)
replicaCount: 1
image.tag: "latest"

Helm 핵심 개념 Key Concepts

Chart

Kubernetes 리소스를 패키징한 단위. templates/, values.yaml, Chart.yaml로 구성. A package of pre-configured Kubernetes resources.

Release

Chart를 클러스터에 설치한 인스턴스. 같은 Chart를 여러 번 설치 가능. 각 Release는 독립적. An instance of a chart running in a cluster.

Go Templates

{{ .Values.key }} 구문으로 values를 참조. if/range/include 등 제어 구문 지원. Template functions with sprig library support.
