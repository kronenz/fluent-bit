Supply Chain Security Layers
🟢 Source: Code review, branch protection, signed commits (GPG)
🟢 Build: Hermetic builds, reproducible, minimal base images (distroless)
🟢 Sign: cosign keyless signing with Sigstore (OIDC-based identity)
🟢 Store: Immutable tags, vulnerability scanning, SBOM attached
🟢 Deploy: Admission controller verifies signature before allowing pod creation
