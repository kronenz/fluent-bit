2. Three Pillars of Observability 관측 가능성의 세 기둥

Metrics (메트릭)

Infrastructure 메트릭 (CPU, Memory, Disk)
Kubernetes 메트릭 (Pod, Node, Deployment)
Custom 메트릭 (StatsD / DogStatsD)
Integration 메트릭 (Redis, PostgreSQL 등)

Logs (로그)

자동 로그 수집 (컨테이너 stdout)
Log Pipeline (파싱/변환/필터)
Log Pattern 자동 감지
Log → Metric 변환 가능

Traces (트레이스)

자동 계측 (dd-trace-java 등)
분산 트레이싱 (서비스 간)
Flame Graph 시각화
Error Tracking 연동
