1
Snapshot Backup / 스냅샷 백업
etcdctl snapshot save /backup/etcd-$(date +%Y%m%d).db
현재 etcd 상태를 파일로 저장합니다. 정기적으로 수행 권장.
2
Verify Snapshot / 스냅샷 검증
etcdctl snapshot status /backup/etcd-20260405.db --write-out=table
해시, 리비전, 키 수, 크기를 확인하여 무결성을 검증합니다.
3
Stop etcd / etcd 중지
systemctl stop etcd
복구 전 etcd 프로세스를 중지합니다.
4
Restore Snapshot / 스냅샷 복구
etcdctl snapshot restore /backup/etcd-20260405.db --data-dir=/var/lib/etcd-restored
스냅샷에서 새 데이터 디렉토리로 복구합니다. 각 노드에서 개별 수행.
5
Start etcd / etcd 재시작
systemctl start etcd
새 데이터 디렉토리로 etcd를 시작합니다. 클러스터가 재형성됩니다.
