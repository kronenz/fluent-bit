Container Security Layers

컨테이너 보안 계층 | Namespace, Cgroup, Seccomp, Capabilities

1. Defense in Depth 심층 방어 계층
Layer 6
AppArmor / SELinux (MAC)
Mandatory Access Control - 파일/네트워크 접근 정책 강제
Layer 5
Seccomp (시스템 콜 필터)
허용된 syscall만 실행 가능 - 커널 공격 표면 축소
Layer 4
Linux Capabilities (권한 세분화)
root 권한을 세분화하여 필요한 것만 부여
Layer 3
Cgroups (리소스 제한)
CPU, Memory, I/O 사용량 제한 - DoS 방지
Layer 2
Namespaces (격리)
PID, Network, Mount, UTS, IPC, User 격리
Rootless Container
비root 사용자로 실행
2. Capability Dropping 권한 제거 (Capability Drop)

ALL Capabilities (전체 권한)

CAP_SYS_ADMIN
CAP_NET_ADMIN
CAP_SYS_PTRACE
CAP_SYS_MODULE
CAP_DAC_OVERRIDE
CAP_SYS_RAWIO
CAP_NET_RAW
CAP_FOWNER
CAP_SETUID
CAP_SETGID
CAP_CHOWN
CAP_NET_BIND_SERVICE
CAP_KILL
➔
drop ALL

Drop Process (제거 과정)

DROP CAP_SYS_ADMIN (mount, 등)
DROP CAP_NET_ADMIN (iptables)
DROP CAP_SYS_PTRACE (디버그)
DROP CAP_SYS_MODULE (커널 모듈)
DROP CAP_DAC_OVERRIDE (파일 권한)
DROP CAP_SYS_RAWIO (하드웨어)
DROP CAP_NET_RAW (raw 소켓)
KEEP 최소 필요 권한만 유지
➔
minimal set

Minimal Set (최소 권한)

CAP_CHOWN
CAP_NET_BIND_SERVICE
CAP_KILL
securityContext: capabilities: drop: ["ALL"] add: ["NET_BIND_SERVICE"] runAsNonRoot: true readOnlyRootFilesystem: true

4. Privileged vs Unprivileged 특권 컨테이너 비교

Privileged Container (특권 컨테이너)

# 절대 프로덕션에서 사용하지 마세요!
docker run --privileged ubuntu

# Kubernetes
securityContext:
  privileged: true
  • 모든 Capability 부여
  • 모든 디바이스 접근 가능 (/dev/*)
  • Seccomp 필터 비활성화
  • AppArmor/SELinux 비활성화
  • 호스트 커널 모듈 로드 가능
  • 호스트 파일시스템 마운트 가능
RISK: CRITICAL / 위험: 치명적

Hardened Container (강화된 컨테이너)

# 프로덕션 권장 설정
docker run \
  --read-only \
  --cap-drop=ALL \
  --cap-add=NET_BIND_SERVICE \
  --security-opt=no-new-privileges \
  --user 1000:1000 \
  myapp

# Kubernetes
securityContext:
  runAsNonRoot: true
  runAsUser: 1000
  readOnlyRootFilesystem: true
  allowPrivilegeEscalation: false
  capabilities:
    drop: ["ALL"]
    add: ["NET_BIND_SERVICE"]
  seccompProfile:
    type: RuntimeDefault
  • 비root 사용자로 실행
  • 읽기 전용 루트 파일시스템
  • 최소 Capability만 부여
  • 권한 상승 차단
  • Seccomp 기본 프로파일 적용
  • AppArmor 기본 프로파일 적용
RISK: LOW / 위험: 낮음
