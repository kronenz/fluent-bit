# 06. 연봉 협상 전략 — 73M 기반 협상 시나리오

> **TL;DR**
> - 현재 연봉 7,300만 원(인센티브 포함 7,500만+). 3.3% 원천징수 사업소득자라면 **실수령 기준 15-20% 이상** 올려야 실질 이득이다.
> - 목표 제시 금액: **8,500만 ~ 9,000만 원**. 최소 수용선: 8,000만 원.
> - 숫자만이 아니라 **성장 기회, 복리후생, 스톡옵션**을 종합적으로 협상한다.

> **면접 빈출도**: ★★★
> **예상 소요 시간**: 20min

## 현재 보상 분석

### 기본 구조

| 항목 | 금액 | 비고 |
|---|---|---|
| 연봉 (기본) | 7,300만 원 | 세전 |
| 인센티브 포함 | 7,500만+ | 변동 가능 |
| 계약 형태 | 확인 필요 | **3.3% 사업소득 vs 4대보험 근로소득** |

### 3.3% 사업소득자 고려사항

만약 현재 3.3% 원천징수 계약이라면, 이직 시 정규직(4대보험)으로 전환된다. 이 경우:

```
3.3% 사업소득 7,300만 원 실수령
= 약 7,060만 원 (3.3% 원천징수 후)
+ 4대보험 미가입 (본인 부담 없음)
+ 퇴직금 없음
+ 유급휴가 보장 없음

정규직 7,300만 원으로 이직하면?
= 약 5,350만 원 실수령 (소득세 + 4대보험 공제)
+ 4대보험 가입 (건강보험, 국민연금 등)
+ 퇴직금 적립 (~8.3%, 약 606만 원/년)
+ 유급휴가 보장

∴ 동일 금액 이직 시 실수령 약 1,700만 원 감소
  (단, 퇴직금 + 4대보험 혜택으로 일부 상쇄)
```

### 실질 동등 연봉 계산

```
현재 실질 수령: ~7,060만 원 + α(자유도)
정규직 동등 실수령을 맞추려면:

세전 연봉 약 8,500만 원 필요
(실수령 약 6,100만 원 + 퇴직금 ~706만 원 = 실질 ~6,800만 원)

※ 이는 "똑같이 맞추는" 수준이고,
  이직의 기회비용을 고려하면 +10-15%가 합리적
```

|---|---|---|

| 5-8년 | 7,500만 ~ 10,000만 | 시니어 DevOps |
| 8-10년+ | 9,000만 ~ 13,000만 | 리드/아키텍트급 |
| AI/ML 인프라 특화 | +10-20% 프리미엄 | 수요 > 공급 |

| 항목 | 일반적 범위 |
|---|---|
| 기본 연봉 | 시장가 80-100% |
| 스톡옵션/RSU | 연봉의 10-30% 가치 |
| 사이닝 보너스 | 0 ~ 500만 원 |
| 성과 보너스 | 연봉의 5-15% |
| 기타 | 식대, 장비, 교육비, 원격근무 등 |

### Allganize 특수 요인

- **B2B SaaS**: 매출 안정성이 높아 연봉 지급 능력 양호

## 협상 전략

### 금액 프레이밍

```
제시 전략: 앵커링 효과 활용

희망 연봉 질문 시:
"현재 인센티브 포함 7,500만 원 수준이고,
 이직에 따른 기회비용과 성장 기회를 고려해서
 8,500만 원에서 9,000만 원 사이를 희망합니다."

※ 높은 쪽(9,000만)을 먼저 말하면 앵커링 효과
※ 범위로 제시하면 협상 여지가 생김
```

### 타이밍 전략

```
면접 초반에 연봉 질문이 오면:
"구체적인 금액은 서로 역할과 기대치를 충분히 공유한 후에
 논의하면 좋겠습니다."
→ 면접 후반으로 미루는 것이 유리 (가치 증명 후 협상)

면접 후반에 오면:
→ 준비된 금액으로 자신 있게 제시

서면/이메일 오퍼 후:
→ 가장 유리한 타이밍. 이미 합격 결정된 상태이므로 협상력 최대
```

## 시나리오별 대응 스크립트

### 시나리오 1: "현재 연봉이 어떻게 되세요?"

```
현재 인센티브 포함 약 7,500만 원 수준입니다.
다만 계약 형태가 다를 수 있어서,
정규직 전환 기준으로 8,500만 원에서 9,000만 원 사이를 희망합니다.
```

**포인트**: 현재 연봉을 솔직하게 밝히되, 바로 희망 금액으로 연결

### 시나리오 2: "희망 연봉이 좀 높은데요"

```
제가 그 금액을 말씀드린 이유가 있습니다.

첫째, 현재 계약 형태에서 정규직으로 전환하면
실수령 기준으로 상당한 차이가 발생합니다.
동일 수준을 맞추려면 8,000만 원 이상이 필요합니다.

둘째, 10년 이상의 인프라 경험과
AI 데이터센터 도메인 지식은
시장에서 프리미엄이 있는 역량이라고 생각합니다.

다만 연봉만이 전부가 아니라고 생각해서,
스톡옵션이나 성장 기회 등 종합적으로 논의할 수 있으면 좋겠습니다.
```

**포인트**: 근거 제시 + 유연성 표현

### 시나리오 3: "8,000만 원까지는 가능합니다"

```
감사합니다. 8,000만 원이면 기본 연봉 기준으로 이해하면 될까요?

그러면 추가로 몇 가지 확인하고 싶은 부분이 있습니다.
- 성과 보너스 체계는 어떻게 되나요?
- 스톡옵션은 제공되나요? 있다면 규모는?
- 교육/자격증 지원 예산이 있나요?
- 장비 지급 기준은?

종합 패키지 기준으로 판단하고 싶습니다.
```

**포인트**: 기본 연봉이 기대보다 낮으면 복리후생으로 보완

### 시나리오 4: "연봉은 입사 후 성과로 증명해주세요"

```
그 말씀도 이해합니다.
다만 이직에 따른 기회비용이 있기 때문에,
최소한 현재 보상 수준은 보장되어야
저도 안심하고 업무에 집중할 수 있을 것 같습니다.

수습 기간 후 성과 기반 조정이 있다면,
어떤 기준과 절차로 진행되는지 알 수 있을까요?
```

**포인트**: "증명하겠다"에 끌려가지 않되, 대립하지도 않기

## 협상 체크리스트

### 금전적 항목

- [ ] 기본 연봉 (세전)
- [ ] 성과 보너스 (비율, 지급 기준)
- [ ] 사이닝 보너스 유무
- [ ] 스톡옵션/RSU (수량, 행사가, 베스팅 기간)
- [ ] 퇴직금 별도 적립 여부

### 비금전적 항목

- [ ] 근무 형태 (출퇴근 / 하이브리드 / 원격)
- [ ] 장비 지급 (노트북 사양, 모니터 등)
- [ ] 교육/자격증 지원 (금액, 범위)
- [ ] 컨퍼런스 참가 지원
- [ ] 유급 휴가 일수
- [ ] 식대/교통비
- [ ] 건강검진

### 성장 관련

- [ ] 온보딩 프로그램 유무
- [ ] 멘토링 체계
- [ ] 사내 스터디/기술 공유 문화
- [ ] 커리어 성장 경로 (IC vs 매니지먼트)
