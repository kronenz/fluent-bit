핵심 차이점 Key Differences

HTTP/1.1 - Head-of-Line Blocking

각 TCP 연결에서 한 번에 하나의 요청만 처리. 브라우저는 보통 도메인당 6개 연결을 열어 병렬화 시도.

HTTP/2 - Stream Multiplexing

하나의 TCP 연결에서 여러 스트림을 동시에 전송. 하지만 TCP 레벨 HOL blocking은 여전히 존재.

HTTP/3 - QUIC Protocol

UDP 기반 QUIC 프로토콜 사용. 스트림이 완전히 독립적이어서 패킷 손실이 다른 스트림에 영향 없음. 0-RTT 연결 재개 지원.

Transport Layer 전송 계층

HTTP/1.1, HTTP/2: TCP 사용 | HTTP/3: UDP + QUIC 사용. QUIC은 TLS 1.3을 내장하여 핸드셰이크 시간 단축.
