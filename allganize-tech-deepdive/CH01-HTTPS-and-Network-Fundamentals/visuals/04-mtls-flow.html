3. 전통적 보안 vs Zero Trust 비교

🏘 전통적 경계 보안 (Perimeter Security)

🚫

네트워크 경계(방화벽)만 보호 - 내부는 신뢰

🚫

VPN으로 접속하면 내부 리소스 모두 접근 가능

🚫

내부 트래픽 암호화 없음 - 내부 공격에 취약

🚫

Lateral Movement 공격에 무방비

🚫

"Castle and Moat" 모델 - 성벽만 뚫으면 끝

🔒 Zero Trust 보안 모델

✅

"Never trust, always verify" - 모든 요청 검증

✅

mTLS로 모든 서비스 간 통신 암호화 + 인증

✅

최소 권한 원칙 - RBAC/ABAC으로 세밀한 접근 제어

✅

마이크로세그멘테이션 - 서비스별 네트워크 격리

✅

Istio AuthorizationPolicy로 L7 레벨 접근 제어
